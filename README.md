### 🛠 得意分野

- 🤖 **ブラウザ操作の自動化** — フォーム入力・データ収集・定型操作（Playwright）
- 📊 **Excel / CSV の自動処理** — 集計・転記・突合・整形（openpyxl / pandas）
- 🖼 **PDF・画像処理 / 画像認識** — 加工・変換・物体検出（PyMuPDF / Pillow / YOLO）
- 🌐 **Webツール開発・デプロイ** — Flask / Docker / Hugging Face Spaces

### 🚀 代表作 — 帯付けくん（obiduke-kun）

不動産チラシ（マイソク）に載った**他社の帯・ロゴをAI（YOLOv8）で自動検出**し、除去・自社の帯に差し替えるWebツール。手作業で数分かかる加工が数秒で終わります。
